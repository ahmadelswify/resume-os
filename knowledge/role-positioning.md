## How to Use This File

For each role type you target, describe:
1. **Lead with:** What should come first in the summary and skills?
2. **Emphasize:** Which achievements/metrics should be front and center?
3. **De-emphasize:** What's less relevant for this role type?
4. **Keywords:** Industry-specific terms to include
5. **Framing angle:** How to position yourself (builder, leader, strategist, etc.)
